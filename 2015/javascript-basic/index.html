变量

Javascript是一门解释型的动态语言。所谓解释型，区别于C/C++/Java等编译型语言，并不会在运行将源代码编译为类文件，而是在运行时动态解析，边解释边运行。所谓动态，即是在定义变量时，不用强制定义变量的类型，而是在执行时，动态确定变量的类型。动态语言的特性，确定了Javascript这门语言随性的气质，也是Javascript的灵动飘逸所在。理解Javascript的精髓，首先应该理解Javascript中的变量定义。

声明与定义

var a = 1;

这是一句最简单的变量定义。而实际上基本等价于做了两个操作：定义变量a，然后将数值1复制给变量a。

var a;
a = 1;

var a; 称为变量的声明。如果只是对变量声明，而未赋初始值，则默认的值为 undefined。

a = 1;为对变量的赋值。将数值1赋值给变量a。

在JavaScript中，允许省略var，直接对未声明的变量赋值。也就是说，var a = 1 与 a = 1，这两条语句的在执行时的效果看上去是相同的。

但是，需要特别注意的是，如果对变量未声明而直接赋值的话（如 a = 1;），此时的变量将定义为全局变量（即便语句在函数内部）！此时往往会不知不觉的造成全局变量污染。

原则1：记得变量要先声明再赋值，不要忘记使用var。
